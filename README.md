# Page 369

This project demonstrates a one-page (also called single-page) website built with [Bootstrap 5](https://getbootstrap.com/).

## Installation

Clone this repository to your local computer:

```bash
git clone https://github.com/contaware/page369.git
```

## Adaptation

Just change *index.html* to your needs. The pictures are located in the *img/* directory.

You can find useful notes in [our bootstrap repository](https://github.com/contaware/html-css-docs-notes/tree/main/bootstrap).
